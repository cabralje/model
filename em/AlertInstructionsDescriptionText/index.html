em:AlertInstructionsDescriptionText

element AlertInstructionsDescriptionText in namespace http://release.niem.gov/niem/domains/emergencyManagement/4.1/

Definition

A description of the recommended action to be taken by recipients of the alert message. An extended human readable instruction to targeted recipients. If different instructions are intended for different recipients, they should be represented by use of multiple info blocks
