em:AlertEventDetailsHeadlineText

element AlertEventDetailsHeadlineText in namespace http://release.niem.gov/niem/domains/emergencyManagement/4.1/

Definition

A headline of the alert message. A brief human-readable headline. Note that some displays (for example, short messaging service devices) may only present this headline; it SHOULD be made as direct and actionable as possible while remaining short. 1
