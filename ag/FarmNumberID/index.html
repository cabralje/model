ag:FarmNumberID

element FarmNumberID in namespace http://release.niem.gov/niem/domains/agriculture/4.0/

Definition

An identifier assigned to all land under control of a particular operator within a particular county. Each County assigns Farm Numbers independently. As a result, farms in different counties might be assigned the same Farm Number; i.e, a Farm Number might be unique only within a particular county. Thus, to uniquely identify a farm, the State FSA Code and County FSA Code must be specified in combination with the Farm Number.
