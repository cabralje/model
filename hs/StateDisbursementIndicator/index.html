hs:StateDisbursementIndicator

element StateDisbursementIndicator in namespace http://release.niem.gov/niem/domains/humanServices/4.1/

Definition

True if the child-support order directs the obligor to make payments to a state agency for disbursement to the custodial parent; false otherwise.
