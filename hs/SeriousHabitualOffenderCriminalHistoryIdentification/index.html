hs:SeriousHabitualOffenderCriminalHistoryIdentification

element SeriousHabitualOffenderCriminalHistoryIdentification in namespace http://release.niem.gov/niem/domains/humanServices/4.1/

Definition

An identification that is used to link a criminal history record to a person. The identification is sometimes called a file control number (FCN).
