hs:PersonUnionAtBirthIndicator

element PersonUnionAtBirthIndicator in namespace http://release.niem.gov/niem/domains/humanServices/4.1/

Definition

True if a child's biological parents were in a Person Union with each other at the time of the child''s birth; false otherwise.
