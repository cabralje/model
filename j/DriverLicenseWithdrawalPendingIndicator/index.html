j:DriverLicenseWithdrawalPendingIndicator

element DriverLicenseWithdrawalPendingIndicator in namespace http://release.niem.gov/niem/domains/jxdm/6.1/

Definition

True if a current licensing jurisdiction has an action pending that, if carried to completion, will cause a withdrawal of a specified driver privilege type; false otherwise.
