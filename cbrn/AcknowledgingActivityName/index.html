cbrn:AcknowledgingActivityName

element AcknowledgingActivityName in namespace http://release.niem.gov/niem/domains/cbrn/4.1/

Definition

A name of the activity that produced an acknowledgement. This property can be used to provide data for workflow coordination by the sending or receiving systems.
