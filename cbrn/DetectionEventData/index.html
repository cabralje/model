cbrn:DetectionEventData

element DetectionEventData in namespace http://release.niem.gov/niem/domains/cbrn/4.1/

Definition

A set of all data collected during a Detection Event that involved the inspection of an Item(s) for the purpose of detecting the presence of illicit goods and materials. This includes data collected by the device(s) used to perform the detection as well.
