cbrn:SpectrumPeakNetAreaValue

element SpectrumPeakNetAreaValue in namespace http://release.niem.gov/niem/domains/cbrn/4.1/

Definition

A value for the net number of counts in a peak; i.e., total counts minus continuum counts. No other adjustment (e.g., environmental background subtraction), should be performed.
