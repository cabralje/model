cbrn:AudioFileCodecName

element AudioFileCodecName in namespace http://release.niem.gov/niem/domains/cbrn/4.1/

Definition

A name of the codec used to create the audio file ex. mpga MPEG audio (recommended for portability) mp3 MPEG Layer 3 audio mp4a MP4 audio a52 Dolby Digital (A52 or AC3) vorb Vorbis spx Speex flac or fl32 FLAC
