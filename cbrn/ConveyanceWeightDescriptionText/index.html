cbrn:ConveyanceWeightDescriptionText

element ConveyanceWeightDescriptionText in namespace http://release.niem.gov/niem/domains/cbrn/4.1/

Definition

A description of conveyance weight, such as gross weight, axle weight, etc., this element is used to provide the description that is applicable to the value provided by the Conveyance weight measure.
