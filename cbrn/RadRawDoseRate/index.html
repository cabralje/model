cbrn:RadRawDoseRate

element RadRawDoseRate in namespace http://release.niem.gov/niem/domains/cbrn/4.1/

Definition

A DoseRate measurement data element(s) used to produce derived data. There shall be no duplicate IDREF values in the list. This is required whenever the element is used within a DerivedData block, but is prohibited otherwise.
