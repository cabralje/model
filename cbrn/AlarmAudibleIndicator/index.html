cbrn:AlarmAudibleIndicator

element AlarmAudibleIndicator in namespace http://release.niem.gov/niem/domains/cbrn/4.1/

Definition

True if an audible alarm was announced; false otherwise. An indicator if an audible alarm was annunciated.
